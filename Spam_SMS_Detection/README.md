
# Spam SMS Detection

## Objective
Develop a machine learning model to classify SMS messages as Spam or Ham (Not Spam).

## Dataset
CodSoft Spam SMS Dataset (spam.csv)

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Algorithm
- TF-IDF Vectorization
- Multinomial Naive Bayes

## Features
- Data Preprocessing
- Text Cleaning
- TF-IDF Feature Extraction
- Model Training
- Spam Prediction
- Confusion Matrix
- Classification Report
- Accuracy Evaluation

## Files Included
- Spam_SMS_Detection.ipynb
- spam_sms_model.pkl
- requirements.txt
- README.md

## Results
- Successfully classified SMS messages into Spam and Ham.
- Achieved high accuracy using the Multinomial Naive Bayes classifier.
