# CODSOFT Machine Learning Internship

This repository contains the projects completed as part of the CodSoft Machine Learning Internship.

## Projects

### 1. Credit Card Fraud Detection
- Detect fraudulent credit card transactions using Machine Learning.

### 2. Customer Churn Prediction
- Predict whether a customer is likely to leave using Machine Learning.

### 3. Spam SMS Detection
- Classify SMS messages as Spam or Ham using Natural Language Processing (NLP).

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
